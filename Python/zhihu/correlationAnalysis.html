# excel实现

步骤：

  1. 文件
  2. 选项
  3. 加载项
  4. ‘管理’这里选择‘Excel加载项’
  5. ‘转到’
  6. 选择‘分析工具库’----这就完成了excel的数据分析功能了
  7. 在选项栏里，选择‘数据’，靠右侧有‘数据分析’
  8. 选择‘相关分析’
  9. 寄几操作去

额，最后得到的样式，如别人论文的结论的格式

# Excel绘图

步骤：

  1. 选择数据，包括列名
  2. 选项栏，选择‘插入’--->‘推荐的图表’
  3. 选中生成的图表
  4. 工具栏‘插入’--->‘文本框’
