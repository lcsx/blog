# 四分位数

得先了解什么是 中位数 中间的数

四分位数（Quartile）也称四分位点，是指在统计学中把所有数值由小到大排列并分成四等份，处于三个分割点位置的数值。

四分位数是通过3个点将全部数据等分为4部分，其中每部分包含25%的数据。

中间的四分位数就是中位数，因此通常所说的四分位数是指处在25%位置上的数值（称为下四分位数）和处在75%位置上的数值（称为上四分位数）

一般： Q1的位置= (n+1) × 0.25 Q2的位置= (n+1) × 0.5 Q3的位置= (n+1) × 0.75

中度异常k=1.5 极度异常k=3

# pd带数据框的，四分位操作

df = pd.read_csv('./data/data/HR.csv')
# print(df)
# 异常值分析
# 数据里有没有null
# isnull() 方法
le_s = df['last_evaluation']

# 去掉异常值的方法2，推荐
# 理论
# 异常值的定义：
# 四分位间距=上四分位数-下四分位数
# 上四分位数+1.5到3倍的间距，可以确定一个上界
# 下四分位数+1.5到3倍的间距，可以确定一个下界
# 上下界所确定的间距，可认为“正常值”
# 上下界之外的，可认为是“异常值”

# 下四分位数quantile
q_low = le_s.quantile(q=0.25)
# 上四分位
q_high = le_s.quantile(q=0.75)
# 四分位间距
q_interval = q_high - q_low
# print(q_interval)
#0.30999999999999994

# 开始筛选
# 设置倍数
k = 1.5
le_s = le_s[le_s<q_high+k*q_interval][le_s>q_low-k*q_interval]   # 同时满足两个条件
# print(le_s)
# len(le_s)

# numpy 四分位

import numpy as np

le_s = np.array([69,69,70,70,70,70,71,71,71,72,73,300])

#下四分位数
lower_q=np.quantile(le_s,0.25,interpolation='lower')

#上四分位数
higher_q=np.quantile(le_s,0.75,interpolation='higher')

#四分位距
int_r=higher_q-lower_q

k = 1.5

# 上界 = 上四分位数 + 1.5到3倍的间距
upper = higher_q + k * int_r

# 下界 = 下四分位数 + 1.5到3倍的间距
lower = lower_q - k * int_r

le_s = le_s[ (le_s>lower) & (le_s<upper) ]
