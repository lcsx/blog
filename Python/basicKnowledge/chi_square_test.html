    # chi-square test 卡方检验

    卡方检验属于非参数检验，主要是比较两个及两个以上样本率（构成比）以及两个分类变量的关联性分析。

    根本思想在于比较理论频数和实际频数的吻合程度或者拟合优度问题。

    # 例

    # 卡方检验
# 生成标准正太分布的数
# 检测数列是否符合正态分布


import numpy as np
import scipy.stats as ss

# 生成20个符合标准正太分布的数
norm_dist = ss.norm.rvs(size=20)
print(norm_dist)

# 检验一个数列是否符合标准正太分布
# print(ss.normaltest(norm_dist))
# NormaltestResult(statistic=6.885301147102693, pvalue=0.03197980805741496)

# 卡方检验
list1 = [[15,95],[85,5]]
print(ss.chi2_contingency(list1))
# (126.08080808080808, 2.9521414005078985e-29, 1, array([[55., 55.],[45., 45.]]))
# 顺序 (检验统计量，p值，自由度，理论分布)
