# z检验

数据分析中有一块很大的版图是属于均值对比的，应用广泛。例如，对比试验前后病人的症状，证明某种药是否有效；对比某个班级两次语文成绩，验证是否有提高；对比某个产品在投放广告前后的销量，看广告是否有效。这些都属于两均值对比的应用。

均值对比的假设检验方法主要有Z检验和T检验，它们的区别在于Z检验面向总体数据和大样本数据，而T检验适用于小规模抽样样本。

# z检验与t检验对比

检验 属于
z 均值对比 Z检验面向总体数据和大样本数据 要求总体标准差已知或者样本容量足够大
t 均值对比 T检验适用于小规模抽样样本 只需从正态总体中抽取小规模的样本数据，并计算均值与标准差，用来代替正态总体的均值和标准差即可

# 显著性水平

显著性水平是估计总体参数落在某一区间内，可能犯错误的概率，用α表示。

α表示原假设为真时，拒绝原假设的概率。 估计总体参数落在某一区间内，可能犯错误的概率为显著性水平，用α表示。 1-α 为置信度或置信水平，其表明了区间估计的可靠性。

通常取α=0.05或α=0.01。这表明，当作出接受原假设的决定时，其正确的可能性（概率）为95%或99%。

把要检验的假设称作原假设，记作H0，把与H0对立的假设称作备择假设，记作H1。

# 3σ原则

  • 约有 68.2% 的点落在 ±1 个标准差的范围内
  • 约有 95.5% 的点落在 ±2 个标准差的范围内
  • 约有 99.7% 的点落在 ±3 个标准差的范围内

# python检验是否为正太分布

import numpy as np
import scipy.stats as ss

# 随机生成一个符合标准正太分布的20个数
norm_dist = ss.norm.rvs(size=20)
print(norm_dist)
a = [1,2,3,4,5,6,7,18,9,20,1,2,33,4,5]
# 如何检测是否是正太分布呢
res1 = ss.normaltest(a)   
print(res1)
# NormaltestResult(statistic=0.9547603432197074, pvalue=0.6204066234691221)
# 如果显著性水平是0.05的话 pvalue  p值大于0.05则表示符合正太分布
