# 分组分析
# 钻取是改变维的层次，变换分析的粒度
# 向上钻取 如知道每天销售额，汇总成每个月的销售额，已月的单位的分析，这个过程是向上钻取
# 向下钻取 （展开数据，查看数据的过程）比如考试，每个班都是一个分组，知道每个班级成绩，想看
# 每个班的男生女生成绩的过程，就是向下钻取
