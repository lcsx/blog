# 异常

throwable类有两个重要子类：

  1. Error：程序无法处理的错误，如虚拟机错误、内存溢出。不用去处理
  2. Exception：程序本身可以处理的异常。

# 1.1 异常处理机制

处理机制为：抛出异常，才能捕获异常。

对于运行时异常、错误或可查异常，java对异常处理方式不同。

  1. 对可查异常必须捕获、或者声明抛出。
  2. 循序忽略不可插的RuntimeException(含子类)和Error(含子类)。

# 1.2 try-catch-finally

try块后可接0个或多个catch块，如果没有catch块，则必须跟一个finally块。

import java.util.Scanner;

public class Test1 {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        System.out.println("====运算来说====");
        try{
            System.out.print("请输入第一个整数");
            int one = input.nextInt();
            System.out.print("请输入第二个整数");
            int two = input.nextInt();
            System.out.print("商是："+(one/two));
        }catch (Exception e){
            System.out.println("程序出错了");
            e.printStackTrace(); //输出错误信息
        }finally {
            System.out.println("====运行结束====");
        }
    }
}

# 1.5 异常处理中的return

若finally中有return，try、catch中的return都不会被执行。

import java.util.Scanner;

public class Test1 {
    public static void main(String[] args) {
        int res = get1();
        System.out.println(res);
    }

    public static int get1(){
        Scanner input = new Scanner(System.in);
        System.out.println("====运算开始====");
        try{
            System.out.print("请输入第一个整数");
            int one = input.nextInt();
            System.out.print("请输入第二个整数");
            int two = input.nextInt();
            return one/two;
        }catch (Exception e){
            System.out.println("出错了");
            return 0;
        }finally {
            System.out.println("====运行结束====");
            return 100000;
        }
    }
}

# 2 thowrs

通过throws声明将要抛出何种类型的异常，通过throw将产生的异常抛出。

如果一个方法可能会出现异常，但没有能力处理这种异常，可以再方法声明处用throws子句来声明抛出异常。谁调用这个方法，谁来处理这个异常。

throws语句用在方法定义时声明该方法要排除的异常类型

pubclic void method() throws Exception1,Exception2,...,ExceptionN{
	//可能产生异常的代码
}

当方法排除异常列表中的异常时，方法将不对这些类型及其子类型的异常作处理，而抛向调用该方法的方法，由ta去处理。

# 3 throw

throw用来抛出一个异常，如 throw New IOException();

throw抛出的只能是可抛出类Throwable或者子类的实例对象。

throw抛出异常通常由两种方案：

  1. throw抛出的异常，自己处理
  2. 在抛出异常处，用throws标识异常类型。谁调用方法，谁处理异常。

# 3.1 throw抛出的异常，自己处理

# 4 自定义类型

自定义异常，就是定义一个类，去继承Throwable类或者ta的子类。

package imooc.lcs.test;

public class HException extends Exception{
    public HException(String str){
        super(str);
    }
}


import imooc.lcs.test.HException;

import java.util.Scanner;

public class Test1 {
    public static void main(String[] args) {
        try {
            testAge();
        } catch (HException e) {
            System.out.println(e.getMessage());
            System.out.println("酒店前台工作任意不允许办理入住登记");
        } catch (Exception e) {
            e.printStackTrace();
        }
    }

    //描述酒店的入驻规则：限定年龄，18岁以下、80岁以上的住客必须有亲友陪伴
    public static void testAge() throws Exception{
        System.out.println("请输入年龄：");
        Scanner input = new Scanner(System.in);
        int age = input.nextInt();
        if(age < 18 || age > 80){
            throw new HException("18岁以下，80岁以上的住客必须有亲友陪伴");
        }else{
            System.out.println("欢迎入驻本酒店");
        }
    }
}

# 5 异常链

捕获异常后，再抛出另一个异常。且保留前面异常的信息。

package imooc.lcs.test;

public class HException extends Exception{
    public HException(String str){
        super(str);
    }
}

import imooc.lcs.test.HException;

public class Test1 {
    public static void main(String[] args) {
        try{
            testThree();
        }catch (Exception e){
            e.printStackTrace();
        }
    }

    public static void testOne() throws Exception{
       throw new HException("lcs123");
    }

    public static void testTwo(){
        try{
            testOne();
        }catch (HException e){    //异常111
            // throw抛出异常，用方法1.自己处理
            try {
                throw new Exception("辛的异常1",e);  //保留前面异常信息方法1。   异常111 的信息
            } catch (Exception exception) {
                exception.printStackTrace();
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
    }

    public static void testThree() throws Exception {
        try {
            testTwo();
        }catch (Exception e){
            //throw new Exception("辛的异常2");
            //保留异常信息方法2
            Exception e1 = new Exception("辛的异常2");
            e1.initCause(e);
            throw e1;

        }
    }
}
