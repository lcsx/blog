# 包装类

java数据类型分为 基本数据类型 和 引用数据类型。

# 基本数据类型是不具有对象特征的，即没有 属性 和 方法、对象化的交互。

# 1 包装类作用

包装类就是让基本数据类型具有属性、方法，可以对象化交互。

|基本类型 |对应的引用类型 | |boolean |java.lang.Boolean | |byte |java.lang.Byte | |short |java.lang.Short | |int |java.lang.Integer | |long |java.lang.Long | |float |java.lang.Float | |double |java.lang.Double | |char |java.lang.Character|

# 2 装箱与拆箱 & 基本数据类型的装箱与拆箱

# 2.1 装箱

装箱：把 基本数据类型 装换成 包装类

装箱有两种方法：

  1. 自动装箱
  2. 手动装箱

自动装箱：

int t1 = 2;
Integer t2 = t1;

手动装箱：

Integer t3 = new Interger(t1);

# 2.2 拆箱

拆箱：把 包装类 转换成 基本数据类型

也分

  1. 自动拆箱
  2. 手动拆箱

自动拆箱：

int t4 = t2;

手动拆箱：

int t5 = t2.intValue();
doubel t6 = t2.doubleValue();

# 3 基本数据类型转字符串

# 3.1 基本数据类型字符串

int t1 = 2;
String t2 = Integer.toString(t1);

# 3.2 字符串转基本数据类型

# 3.2.1 包装类的parse

int t3 = Integer.parseInt(t2);

# 3.2.2 包装类的valueOf

int  t4 = Integer.valueOf(t2);

# 5

// 手动装箱
Integer one = new Integer(100);
Integer two = new Integer(100);
System.out.printIn("one==two的结果"+(one == two));  //false
// new 开辟新的内存空间
// == 内存空间和值相同才为true
// 因为one、two都用new，所以说false

//自动拆箱
Integer three = 100;
System.out.printIn("three==100的结果"+(three == 100));  //true  这里有自动拆箱了

//Integer four = 100;
//实际上，上一句执行的是
Integer four = Integer.valueOf(100);
System.out.printIn("three==100的结果"+(three == four)); //true

以上对象常量池概念，除了对FFloat、Double包装类外，都适用。

即

Double d1 = Doubel.valueOf(100);
System.out.printIn("d1==100的结果"+(d1 == 100)); //true
