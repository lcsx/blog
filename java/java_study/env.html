# 逻辑运算符

包括五种：

  1. &&
  2. &
  3. ||
  4. |
  5. !

“ && ”、“ || ”是短路运算符，只要第一个表达式的值就能决定表达式最后的结果，运算符右边的表达式就不执行了。

但“&”、“|”都会执行双边表达式

public class ADAS {
    public static void main(String[] args) {
        int n = 3;
        boolean b = (3<7)||((n++)<3);
        System.out.println(b);
        System.out.println(n);
    }
}
// true 3    
// n++没有执行

# 冒泡排序

  1. 比较相邻的元素。如果第一个比第二个大，就交换他们两个。
  2. 对每一对相邻元素做同样的工作，从开始第一对到结尾的最后一对。在这一点，最后的元素应该会是最大的数。
  3. 针对所有的元素重复以上的步骤，除了最后一个。
  4. 持续每次对越来越少的元素重复上面的步骤，直到没有任何一对数字需要比较。

比如从小到达排，N个元素，每一轮都能排出这轮的最大值，所以外循环只需要N-1轮就能排好。

排好一轮，就忽略一个元素，所以在内循环中每轮都会少一个元素。

public class ADAS {
    public static void main(String[] args) {
        int temp;
        int[] a = {11,2,31,5};
        for(int i=0;i<a.length-1;i++){
            for(int j=0;j<a.length-i-1;j++){
                if(a[j]>a[j+1]){
                    temp = a[j];
                    a[j] = a[j+1];
                    a[j+1] = temp;
                }
            }
        }
        for(int n:a){
            System.out.println(n);
        }

    }
}

# 二维数组

二维数组的声明 有三种：

int arr[][];

int [][] arr1;

int [] arr2[];

声明时：行数要确定、列数可省略

多维数组的打印

import java.util.Arrays;

public class ADAS {
    public static void main(String[] args) {
        int Arr[][];
        int [][] Arr1;
        int [] Arr2[];

        Arr = new int[3][];
        //直接输出数组是看不到数组的值的，需要先转为字符串
        System.out.println(Arrays.toString(Arr));

        //声明数组的同时进行创建 3*3的数组
        //必须指定行数，列数可以不指定
        int[][] Arr3 = new int[3][];   //也可以直接赋值 int[][] Arr3 = {{1,2},{2,3},{3,4}};

        //每行的列数是可以不一样的
        Arr2 = new int[3][];
        Arr2[0] = new int[2];
        Arr2[1] = new int[3];
        
        //Java.Arrays.deepToString方法可以将多维数组打印出来，实例化后的数组默认值为0，否则为null
        System.out.println(Arrays.deepToString(Arr2));
    }
}

# 求 1!+2!+3!+4!+5!+...+25!的和

public class ADAS {
    
    public static void main(String[] args) {
        //long长整型，int范围小，会溢出
        long sum = 0;
        for(int i=1;i<=25;i++){
            long s = 1;

            sum += s;
        }

}

# 基本类型数据传值与引用类型数据传值

基本类型数据，方法不改变数据的值；

而引用类型，方法操作数据后，会改变原有数据值；

基本数据类型：

public class ADAS {
    public int update(int n){
        return n+1;
    }
    public int xx1(int n){
        int s = update(n)+2;   // 在主方法外，同一个类里的方法调用其他方法，不用声明一个类，直接方法名字调用即可。
                               // 在主方法中调用类里的方法，需要new.
        return s;
    }
    public static void main(String[] args) {
        int lcs = 6;
        ADAS someone = new ADAS();
        System.out.println(someone.xx1(lcs));
        System.out.println(lcs);     //基础类型数据数据，不改变原数据值。
    }
}
# 引用类型的值会被改变
  1. 但若是数据是引用型数据，则对数据的操作会改变原数据值。类似的，对象也会被修改
  2. 返回值是数组类型
  3. 如何打印数组
import java.util.Arrays;

public class ADAS {
    public int[] update(int[] a){        //返回
        a[2] = 9;
        return a;
    }

    public static void main(String[] args) {
        int[] lcs = {1,2,3,4,5};
        ADAS someone = new ADAS();
        int[] b = someone.update(lcs);
        System.out.println(Arrays.toString(b));  //[1, 2, 9, 4, 5]
        System.out.println(Arrays.toString(lcs));//[1, 2, 9, 4, 5]  原数组的值被改变了
    }
}

# 可变参数列表

参数的数量是不确定的。

WARNING

一个方法只能有一个可变参数； 如果函数有多个参数，可变参数要放到最后;

带有可变参数，若是有重复方法名，不是方法重载，而是重新定义；

出现方法重载时，可变参数列表所在的方法是最后被调用的；

public class ADAS {
    public int sum(int... a){
        int res = 0;
       for(int i:a){
           res += i;
       }
       return res;
    }

    public static void main(String[] args) {
        ADAS lcs = new ADAS();
        int x = lcs.sum(1,3,4,5);
        System.out.println(x);
		
		int[] x1 = {1,3,4,5};
        int xx = lcs.sum(x1);   //可变参数 传的是数组
        System.out.println(xx);
    }
}
