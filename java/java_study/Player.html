# 播放器案例

需求：

  1. 播放列表管理
  2. 播放器管理

播放列表管理

  1. 将歌曲添加到主播放列表
  2. 将歌曲添加到普通播放列表
  3. 通过歌曲id查询播放列表中的歌曲
  4. 通过歌曲名称查询播放列表中的歌曲
  5. 修改播放列表的歌曲
  6. 删除播放列表中的歌曲
  7. 显示播放列表中的所以歌曲

播放器管理

  1. 向播放器添加播放列表
  2. 从播放器删除播放列表
  3. 通过名字查询播放列表信息
  4. 显示所有播放列表名称

# 1.1 类

  1. 歌曲类（Song）
  2. 播放列表类（PlayList）
  3. 播放器类（PlayList Collection）
  4. 测试类（Test Demo）

属性：均为字符串类型

  1. 歌曲id（id）
  2. 歌曲名（name）
  3. 演唱者（singer）

方法：

  1. 构造方法
  2. getter和setter方法
  3. hashCode()和equals()方法【播放列表不能用重复歌曲】
  4. toString() 【方便将歌曲信息输出】
