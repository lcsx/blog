# Swing

Swing 是Java新一代的图形界面技术。Swing 是一种轻量级组件，它采用纯 Java 实现，不再依赖于操作系统，所以可以在所有平台上保持相同的运行效果，对跨平台支持比较出色。 除此之外，Swing 提供了丰富的界面组件与便捷的使用方法，是JAVA领域开发界面应用程序的首选技术。本次课程我们就利用Swing提供的各种组件来开发“小慕听书”程序界面。

Swing界面常用组件：JFrame框架窗口、JPanel面板、JButton按钮、JtextField文本框等等，

但通过JFormDesigner设计器，可以通过拖拽的方式直观地使用组件。

# JFormDesigner插件安装与使用

【通过压缩包安装】 在IDEA工具中支持插件扩展，有一款名为JFormDesigner的插件允许开发者使用可视化的方式设计Swing界面，通过这个插件可以极大简化Swing界面开发的难度。

JFormDesigner官网下载地址

需要IDE加载 JFormDesigner7.0.3intellijidea.zip 压缩包，来完成ide安装插件的工作

IDE中，File > Settings > Plugins > 右上齿轮第三项“Install Plugin form Disk...” > 选择JFormDesigner的zip包 > OK

在java项目右键 新建 出现 JFormDesigner Form 证明安装上了。

# JFormDesigner破解

wangpan里有注册机

  1. 解压JFormDesigner的zip，用注册机选择版本，name等随便填。
  2. patch选lib中的“.jar”文件
  3. 点击generate生成JFormDesigner_license.txt文件
  4. 将1步中解压的zip重新打包，然后在ide中安装zip
  5. 新建.jfd文件，断网下，注册选择JFormDesigner_license.txt文件即可

# Create a new Form

【superclass】 选 JPanel（面板）开发界面默认选

【Layouut manager】布局管理器 > 'null LAyout' (很多种布局，类似前端布局) Null布局方案。灵活布局是无任何规则限制的布局方式，开发者可根据自己的意愿任意将界面组件放置在任意位置，一切处理的细节均有程序员自己设计，本次“小慕听书”就采用Null布局进行设计。
