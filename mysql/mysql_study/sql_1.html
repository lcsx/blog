主要内容：

  3. 数据的有条件查询：条件表达式、比较运算符、逻辑运算符、按位运算符

# 1 普通查询

最基本的查询语句由SELECT 和 FROM 关键字组成

# 查询所有字段
SELECT * FROM t_temp;

# 查询某些字段
SELECT empno,ename,sal FROM t_tem;

# 1.1 列别名 AS [也可以不写，用空格隔开]

select子句使用了表达式，可重命名这列名，否则列名就是表达式的名称，如下例即“sal*12”

SELECT empno,sal*12 AS "income" FROM t_emp;
# SELECT empno,sal*12 FROM t_emp;

# 1.2 数据分页 LIMIT

# SELECT ... FROM ... LIMIT 起始位置,偏移量;

SELECT empno,ename FROM t_tem LIMIT 0,5;
# 默认起始位置位0
# 相当 SELECT empno,ename FROM t_tem LIMIT 5;

优先级

优先级 FROME > WHERE > SELECT > ORDER BY > LIMIT

# 1.3 结果集排序 ORDER BY

默认升序 ASC 降序DESC

# SELECT ... FROM ... ORDER BY 列名 [ASC|DESC];
SELECT empno,ename,sal*12 AS "income",mgr FROM t_emp ORDER BY income DESC LIMIT 0,5 ;

多个排序字段,每个排序字段可分别设置升降序

SELECT empno,ename,sal,hiredate FROM t_emp ORDER BY sal DESC,hiredate ASC;

# 1.4 结果集中的重复记录 DISTINCT

# SELECT DISTINCT 字段 FROM ...;

SELECT DISTINCT job FROM t_emp;

DISTINCT去重复记录

DISTINCT 的select子句只能查询一列数据,查询多列去除重复数据 会失效；

DISTINCT 关键字之恶能在Select子句中使用一次

# 2 条件查询

TIP

WHERE子句中，条件执行的顺序是从左到右。

所以，应该把1.索引条件，或2.删掉记录最多的条件写在最左侧。可以提升查询速度。

# 2.1 WHERE

# SELECT ... FROM ... WHERE 条件 [AND | OR ] 条件 ...;

SELECT empno,ename,sal FROM t_emp 
WHERE deptno=10 AND sal >= 2000;

SELECT empno,ename,sal FROM t_emp 
WHERE (deptno=10 OR deptno=20) AND sal >= 2000;

# 2.1 比较运算符

# 2.1.1 >=、!=、IN

表中的数据项有可能为null，如何参与sql语句计算呢.通过IFNULL(字段) 日期计算 日期差,如 DATEDIFF(NOW(),日期_字段) IN包含

# 计算 10+null 、10*null  = null
# 运行
SELECT 10+NULL  #null   

SELECT empno,ename,sal,hiredate
FROM t_emp
WHERE deptno=10 AND (sal+IFNULL(comm,0))*12>15000
AND DATEDIFF(NOW(),hiredate)/365>20;

SELECT 
	empno,ename,sal,deptno,hiredate
FROM t_emp
WHERE deptno IN(10,20,30) AND job!="SALESMAN"
AND hiredate<"1985-01-01";

# 2.1.2 是否为空 IS NULL、范围 BETWEEN、模糊查询 LIKE、正则表达式

表达式 意义 例子
IS NULL 为空 comm IS NULL
IS NOT NULL 不为空 comm IS NOT NULL
BETWEEN .. AND .. 范围 sal BETWEEN 2000 AND 3000
LIKE 模糊查询 ename LIKE "A%"
REGEXP 正则表达式 ename REGEXP "[a-zA-Z]{4}"
# 模糊查询，以A开头  LIKE "A%"
# 含有A字符，LIKE "%A%"
SELECT ename,comm,sal FROM t_emp WHERE comm IS NULL
AND sal BETWEEN 2000 AND 3000
AND ename LIKE "A%";

模糊查询

LIKE模糊查询中，“ % ”是表示任意字符，还有一个 “ _ ”表示占位，

LIKE "_CS" 检索 完整的字符串中，是第一个字符任意字符+CS，的字符串

LIKE "_I%" 检索第一个字符任意，第二个是I，余下任意的 字符串

# 正则查找名字为中文的数据项
SELECT 
	ename,comm,sal
FROM t_emp WHERE 
	ename REGEXP "^[\\u4e00-\\u9fa5]{2,4}$";

正则

^[\U4E00-\U9FA5]{2,4}$ 中，^字符串开头，$字符串结尾

# 2.2 逻辑运算符 AND/ OR /NOT/ XOR

XOR 异或关系，两边都为真或假，即false；一边真，一边假，即true

NOT 字段 IN 跟 字段 NOT IN 一样

SELECT 
	ename,deptno
FROM t_emp
WHERE NOT deptno IN(10,20);   
# WHERE deptno NOT IN(10,20);
